## 欢迎来到我的博客

这是使用 Hugo 和 github-style 主题搭建的个人博客。

### 功能特性

- 🎨 GitHub 风格的简洁设计
- 💬 集成 Gitalk 评论系统
- 🔍 本地搜索功能
- 📱 响应式设计，支持移动端
- 🌙 支持暗色模式
- 📊 支持 LaTeX 数学公式
- 🏷️ 标签和分类支持

<!--more-->

### 开始写作

使用以下命令创建新文章：

```bash
hugo new post/文章标题.md
```

然后在 `content/post/` 目录下编辑生成的文件即可。
